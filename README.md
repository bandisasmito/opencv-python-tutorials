# Create Virtual Environment 
```bash 
python -m venv opencvEnv
.\opencvEnv\Scripts\activate
pip install -r requirements.txt
```
# Manual Install OpenCV Only 
```bash 
pip install opencv-python 
```
# VS Code Version 
Currently using version `1.83.1`
